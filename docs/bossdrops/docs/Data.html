Data

The Data namespace contains data models used by BossDrops:

  • BossData: Implements IData and IHasItemDrops<ItemDropData> (from RisingV.Shared.Data). Represents a boss entry, including:
    • Name: String identifier for the boss.
    • PrefabGuid: The GUID of the boss prefab used to match in-game entities.
    • AssetName: The asset name of the boss.
    • ItemDrops: A collection of ItemDropData (from Core) defining items the boss can drop.
    • LootRollMode, LootCompletionMode, LootSelectionMode, LootShuffleMode: Settings that control how drops are rolled, selected, and distributed.
    • ToString() override for debugging output.

This class encapsulates configuration for each boss's loot behavior.
