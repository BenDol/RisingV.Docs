Managers

The Managers namespace provides the ScriptManager class, responsible for discovering, loading, and managing the lifecycle of scripts. The ScriptManager:

  • Tracks script file paths for each plugin.
  • Detects file changes to trigger automatic reloads (configurable).
  • Compiles and loads scripts into the runtime host via ScriptLoader.
  • Provides methods to reload all or modified scripts, and to initialize and load scripts during plugin startup.

This manager ensures that scripts are properly compiled, loaded, and reloaded as needed at runtime.
